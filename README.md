# Analiza Rynku Nieruchomości w Polsce (2010-2023)

**Projekt na zaliczenie:** Analiza Danych w R i Python (ADRPY2025)  
**Semestr:** 2025/26  
**Temat:** Zróżnicowanie cen nieruchomości między województwami i powiatami 

---

## Przegląd Projektu

Projekt analizuje ceny nieruchomości na polskim rynku w okresie 2010-2023, badając różnice geograficzne, trendy czasowe i wpływ segmentacji rynku (pierwotny/wtórny) na dynamikę cen.
